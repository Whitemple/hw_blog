1. Главная,
2. Обратная связь,
3. Список записей,
4. Страница записи

В домашней работе необходимо:
 - Настроить в линтеры и github actions.

 Приложение будет адаптивным, Mobile-first. Расчитано на 3 типа устройств - смартфоны, планшеты и десктопные компьютеры
